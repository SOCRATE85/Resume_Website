Socrate Quantum Grid Resume Website

This is a futuristic and advanced personal resume website for Khalid, a Cloud Architect. It features a sleek dark theme, dynamic background animations, interactive skill bars, and a responsive design.
Features:

    Quantum Grid Theme: A unique futuristic aesthetic with glowing elements and animations.
    Interactive Skill Bars: Visually represents skill proficiency.
    Dynamic Hero Section: Engaging introduction with animated elements.
    Responsive Design: Optimized for all devices (desktop, tablet, mobile).
    Seamless Navigation: Smooth scrolling to different sections.

Technologies Used:

    HTML5
    Tailwind CSS (via CDN)
    JavaScript
    Google Fonts (Inter, Orbitron)
    Lucide Icons

How to Run Locally:

    Save the index.html file to your computer.
    Open index.html in any modern web browser.
